Project Overview

Proyek freelance bersama tim untuk klien HybridHope. Tim terdiri dari dua orang, dengan saya bertanggung jawab penuh pada implementasi front-end. Klien memberikan desain antarmuka dalam bentuk file PDF, yang kemudian kami implementasikan ke dalam website menggunakan HTML, CSS, dan JavaScript secara native. Proyek difokuskan pada penerapan desain yang konsisten dan sesuai spesifikasi teknis dari klien.
